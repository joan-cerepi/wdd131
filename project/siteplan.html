Project Site Plan

Site Name: Home Chef Haven

Site name was selected because I thought it sounded appropriate for a website where users could easily look up their favorite recipe.

Site Purpose:

The site will use an API to provide users with about 2.3 million recipes they can choose from. Users will be able to look up recipes for making food. No more struggling to come up with something good to have for breakfast, lunch or dinner. Every recipe will be available just a few keystrokes away.

Scenarios:

  • How can I search for recipes that are gluten-free?
  • Is there a way to filter recipes based on their cooking time?
  • Can I search for recipes using specific ingredients?
  • Do you have a category for quick and easy recipes?
  • Can I find recipes suitable for beginners?
  • Is there an option to sort recipes by their level of difficulty?
  • Can I search for recipes by cuisine type?
